Yeah, Why a conversational UI… ?? 🤔

Well, a conversational UI brings a lot of advantages specifically for this use case:

It’s Intuitive : It takes the user out of the user flow. You want to share an image? great, just say so and let me do the rest! You just be you 😎

It’s engaging : I don’t need to sit around and wait for you to start the app. If i find something, I can just let you know with a short and sweet text message ☺️

It’s different : This is a fun way to interact with your phone. Also, chat allows for rich media to be easily integrated with the UI in a manner that is familiar to the user. Instead, I am the one who has to do the learning now!! Nerd mode on! 🤓

As i get to know you better, I will learn to identify which memories you would find interesting, and which jokes make you go ROFLOLOLOL 😂

Anyway… Armed with this new knowledge, I got a makeover! 💇🏻
